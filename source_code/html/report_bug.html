Report a Bug

Severity level 1: Bug does not effect the core functionally of the program. The bug may be something that inconveniences the user or something that a developer should be aware of. If the bug only affects non-supported web browsers (anything other than edge, chrome, or firefox), than the bug is a level 1.

Severity level 2: The bug could possibly or in certain cases affect the core functionally of the program.

Severity level 3: The bug makes part of the program unusable or not usable in the way that the program was intended to work. If this bug in any way causes inaccurate data, then the bug is a level 3.
